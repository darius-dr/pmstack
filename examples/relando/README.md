## How to reproduce

From inside the pmstack plugin directory, with the Relando repos accessible:

```
# In Claude Code, with pmstack installed:
/pm:ingest                  # produces .pmstack/positioning.md
/pm:audience buyers-sellers # produces .pmstack/icp/buyers-sellers.md
/pm:audience brokers
/pm:audience developers
```

The output in your `.pmstack/` directory should match the artifacts in this example folder (modulo timestamps and any newer context in the source MDs).
